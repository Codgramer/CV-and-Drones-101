# CV-and-Drones-101

EEA Winter Project 

This project aims to learn the concepts of Computer Vision through OpenCV in Python. This involves applying various edge and contour detection methods to images along with image filtering(analyzing images by changing the brightness, saturation, and contrast).
Apart from that, it involves Camera Caliberation using a camera matrix and distortion coefficients of a camera so that we can effectively identify and correct image distortions in it.
It also aims to give an idea of ArUco marker detection through a Tello Drone connecting both the marker and the drone through Python code.
